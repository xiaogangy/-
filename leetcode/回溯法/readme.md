## 回溯法

### 回溯本质：
**DFS**，即深度优先遍历。思考过程就是一头扎进去，然后做出选择，继续扎进另外一个递归过程里。

### 三要素
- 路径：已经做出的选择
- 选择列表：当前可以做的选择
- 结束条件：也就是到达决策树底层，无法再做选择的条件

### 套路模板
```JS
    const result = [];
    // 递归的回溯方法
    function backtrack(路径, 选择列表) {
        if 满足结束条件:
            result.add(路径)
        return

        for 选择 in 选择列表:
            做选择
            backtrack(路径, 选择列表)
            撤销选择
        }
```
